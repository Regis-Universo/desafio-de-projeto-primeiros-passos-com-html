O que são Clients?

(voltar)

São os dispositivos que interagem com os usuários, tais como:

  • desktops
  • notebooks
  • tablets
  • celulares
  • etc.

O que são Servers?

(voltar)

São gigantescos computadores que recebem requisições http vindas do client-side e retorna arquivos HTML, CSS, JavaScript, imagens textos, etc.

Como é uma estrutura básica de um HTML?

(voltar)

O HTML é uma Linguagem de Marcação que utiliza tags para construir uma página web.

Entendendo a diferença entre listas ordenadas e não ordenadas

(voltar)

As Listas Ordenadas são utilizadas quando a ordem dos ítens da lista faz diferença

As Listas Não Ordenadas são utilizadas quando a ordem dos ítens da lista não faz diferença
